# Scenario Test Skeleton

This directory is reserved for scenario-level tests driven by the LIBRA spec.

Recommended first scenario names:

- `regular_check_basic_hold`
- `regular_check_conflicting_signals_defer`
- `regular_check_report_needed`
- `push_risk_event_user_decision_required`
- `rebalance_candidate_profit_cost_review`

Keep scenario fixtures close to the test that uses them, or store shared JSON fixtures under `examples/`.
